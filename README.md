# 🎓 POO Java — Travaux Pratiques

Travaux pratiques d'**Initiation à la Programmation Orientée Objet** en Java,
réalisés en 2ème année de Licence Informatique à l'Université de Rouen Normandie (2025-2026).

## 🎯 Objectif

Maîtriser les fondamentaux de la POO en Java et adopter les bonnes pratiques
de développement (Javadoc, conventions de nommage, programmation par contrats).

## 📚 Progression des TPs

| TP | Sujet | Concepts abordés | Statut |
|----|-------|-----------------|--------|
| TP01 | Découverte Eclipse & Java | Syntaxe de base, méthodes | ✅ |
| TP02 | Chiffre de César | Classes, tableaux, encapsulation | 🔄 En cours |
| TP03 | Calculette graphique | Interfaces | ⏳ |
| TP04 | Dés et joueurs | Spécification, composition | ⏳ |
| TP05 | Tamagotchi | Plusieurs classes interagissantes | ⏳ |
| TP06 | Gestion immobilière | Tri, comparaison d'objets | ⏳ |
| TP07 | Morpion | Héritage, polymorphisme | ⏳ |
| TP08 | PyLand 1 | Architecture multi-classes | ⏳ |
| TP09 | PyLand 2 | Composition, héritage avancé | ⏳ |

> **Note TP01 :** réalisé en salle de TP — code non retrouvé, sera revu lors du TP05.

## 🧠 Concepts du cours travaillés

- **Chapitre 1-3** : Objets, classes, encapsulation, références, mutabilité
- **Chapitre 4** : Types de données abstraits (TDA), programmation par contrats, invariants
- **Chapitre 5** : Héritage, polymorphisme, liaison dynamique, interfaces

## 🛠️ Environnement

Java 21 · Eclipse IDE · Git · Linux
